# YouTube Transcripts

This folder contains the YouTube transcript collection workflow for the AI-powered SEO content production research.

## Structure

* `videos.csv` is the source list used for transcript fetching.
* `fetch_youtube_transcripts.py` is the script used to fetch or update transcripts from YouTube captions/transcripts only.
* Expert folders, such as `aleyda-solis/` and `michael-king/`, contain raw YouTube transcript Markdown files.
* `sources.md` is the legacy/source-tracking file kept for now when present.
* Old root-level expert note files were removed because they were outdated and replaced by expert-folder transcript Markdown files.
